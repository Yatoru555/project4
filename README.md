# Мини-RPG на Python / Mini-RPG auf Python

## Описание / Beschreibung

Это простая консольная RPG-игра на Python. Игрок создаёт профиль, исследует уровни, сражается с врагами, получает опыт и золото.  
Es ist ein einfaches Konsolen-RPG-Spiel in Python. Der Spieler erstellt ein Profil, erkundet Levels, kämpft gegen Gegner und sammelt Erfahrung sowie Gold.

Игра демонстрирует работу с функциями, словарями и циклами в Python.  
Das Spiel zeigt den Umgang mit Funktionen, Dictionaries und Schleifen in Python.

---

## Особенности / Features

- Создание и редактирование профиля игрока / Erstellen und Bearbeiten eines Spielerprofils  
- Исследование уровней с постепенно возрастающей сложностью / Erkunden von Levels mit zunehmendem Schwierigkeitsgrad  
- Сражения с врагами (HP, урон, получение опыта и золота) / Kämpfe gegen Gegner (HP, Schaden, Erfahrung und Gold erhalten)  
- Открытие новых уровней после победы / Freischalten neuer Levels nach dem Sieg
